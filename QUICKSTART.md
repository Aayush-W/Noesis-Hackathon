# 🚀 AskMyNotes Quickstart

Follow these 3 simple steps to run the project on your computer.

### Step 1: Get Secrets
Ask your teammate for:
1.  The **GEMINI_API_KEY**.
2.  The **firebase-service-account.json** file.
    - Place the json file inside the `backend/` folder.

### Step 2: Run Setup
Double-click `run_setup.bat` (or run it in your terminal).
- It will install all Python and Node.js dependencies for you.
- When prompted, paste the **GEMINI_API_KEY**.

### Step 3: Launch
Double-click `run_app.bat`.
- This will open two windows (one for the backend, one for the frontend).
- Open **http://localhost:5173** in your browser.

---

### Prerequisites
- [Node.js](https://nodejs.org/) (v18+)
- [Python](https://www.python.org/) (3.11+)
- [MongoDB](https://www.mongodb.com/try/download/community) (Running locally)
